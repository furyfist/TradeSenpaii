# Stock Analysis Project

## Overview
Analysis of Coca-Cola (KO) stock data from 1962 to 2026.

## Data Source
- Provider: Yahoo Finance (yfinance API)
- Date Range: 1962-01-02 to 2026-01-30
- Frequency: Daily

## Database
- PostgreSQL
- Database: stock_analysis_KO
- Table: coca_cola_stock

## Setup Instructions
1. Install PostgreSQL
2. Run `sql/schema.sql` to create tables
3. Import data from `data/raw/coca_cola_stock.csv`
4. Run queries from `sql/queries/`
